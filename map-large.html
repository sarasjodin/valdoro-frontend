The purpose

Valdoro is a fictional retreat and mountain escape created as part of my web development studies at Mid Sweden University.

The project focuses on creating a modern, responsive and user-friendly website inspired by nature, relaxation and countryside experiences.

Technology stack

The website uses a headless WordPress setup with a separate HTML, CSS and JavaScript frontend.

The WordPress backend is hosted on a private VPS server and connected through the REST API.

Advanced Custom Fields (ACF) is used to manage dynamic page content from the WordPress admin panel.
