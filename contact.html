The purpose

Valdoro is a fictional retreat and mountain escape created as part of my web development studies at Mid Sweden University.

The project focuses on creating a modern, responsive and user-friendly website inspired by nature, relaxation and countryside experiences.

Technology stack

The website uses a headless WordPress architecture where WordPress functions as a content management system while the frontend is built separately for improved flexibility and performance.
