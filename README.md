# Valdoro – Retreat & Mountain Escape

Headless WordPress CMS with REST API and frontend built in Vanilla JavaScript.

## About the project

Valdoro is a fictional mountain retreat website developed as a headless WordPress project.
The project focuses on combining a lightweight and visually customized frontend with WordPress as a flexible CMS and backend solution.

The frontend is developed in:

- HTML
- CSS
- Vanilla JavaScript

The backend uses:

- WordPress
- REST API
- Advanced Custom Fields (ACF)
- Custom Post Types (CPT)

Frontend and backend are deployed separately using a headless architecture.
